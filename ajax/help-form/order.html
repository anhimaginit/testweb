how to add or update order

Field Descriptions
Order Title Text The title order. May be change when created warranty
Bill To Select one Click here to ad the person who you want to sent the bill
You can search user in that field. If user is not exist in system. Please click on and fill on that form to create
Cart Multiple Search product in SKU or Name columns.
Enter quantity, price, discount,...
If change product in order, select again on same row.
If remove product from order, click on
Discount Code 1 Code Entering a discount code to apply discount. Accept only one code for one order
Salesman Select one Select Salesman (the Salesman activated on policy holder state)
Payment Subscription Select one subscription if the order pay out multiple times

After fill, click on to save
